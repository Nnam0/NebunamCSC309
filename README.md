# NebunamCSC309

Repository for CSC309 coursework and assignment starter code.

## Summary

This repository contains the starter code and small projects for the CSC309 assignments. It currently includes two Python files:

- `assignment_starter_code.py` — starter code for an assignment (student-supplied)
- `mazeanimate.py` — script for maze animation (student-supplied)

The files are intended to be run with Python 3.8+ on Windows (PowerShell) but should work on other platforms with minor adjustments.

## Prerequisites

- Python 3.8 or newer installed and available on PATH
- (Optional) git for source control

Recommended: work inside a virtual environment.

## Setup (PowerShell)

Open PowerShell and run:

```powershell
# create and activate a venv
python -m venv .venv
.\.venv\Scripts\Activate.ps1

# (optional) install requirements if you have a requirements.txt
if (Test-Path requirements.txt) { pip install -r requirements.txt }
```

## Running

To run the starter code or the maze animation script:

```powershell
# run assignment starter code
python assignment_starter_code.py

# run maze animation
python mazeanimate.py
```

If the scripts accept command-line arguments, check the top of each file or any included documentation for usage details.
